# FAPCOIN API (Vercel-ready)

## Deploying to Vercel

1. Upload this folder (`fapcoin-api`) to a GitHub repo.
2. Go to https://vercel.com and create a new project.
3. Select the repo and deploy.
4. Your API endpoint will be:
   https://<your-vercel-project>.vercel.app/api/locks

This version fetches live SPL token balances for FAPCOIN holders.
Locked balances are placeholders (0) and can be added later.
